Ordember Game





How to Play

Start the Game

Hit Play button to navigate to the game page.

The Goal

Remember the order of the pieces, then tap them in order.

Remember

When the game starts, the number on pieces will be shown for a short period of time. Try to memorize the numbers on piece.

Tap

After the numbers disappear, start tapping the pieces in the order of their numbers.

Scores

For each piece that you tap correctly, you get one point.

End of Game

If you tap all pieces in the correct order, the game ends and you will go to the next level. If you tap a wrong piece, you will loose a heart and will need to repeat the level. If you loose all hearts, the game is over.

Options

Check Sound to turn on sounds.
Press Reset to clear all saved information. Both game data and settings will be removed.
